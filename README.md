## 📌 Overview

This repository contains structured hands-on learning and practical implementations using **Spring Boot and Microservices Architecture**.

It includes classroom tasks, assignments, and mini projects focused on building scalable and production-ready backend applications.

---

## 🛠️ Tech Stack

* Java 17
* Spring Framework
* Spring Boot
* Spring Data JPA
* Hibernate
* Maven
* MySQL
* Docker
* RESTful APIs
* Microservices Architecture

## 🏗️ Architecture Approach

This project follows a layered architecture pattern:

Controller → Service → Repository → Database

Microservices communicate via REST APIs.

---

## ▶️ How to Run

1. Clone the repository
2. Configure `application.properties`
3. Run using Maven:

```
mvn spring-boot:run
```
